# Запуск

## Тестирование

### Обычное
```bash
go test ./...
```

### Покрытие тестами
```bash
go test -cover ./...
```

### С подробным выводом (Рекомендуемый)
```bash
go test -v ./...
```

## Docker
```bash
docker compose up --force-recreate --build -d
```
